# Cashback App

## Stack
- Next.js 14, TypeScript, Tailwind CSS, Prisma

## Design System
- Primary accent: #00E88F
- Background: #0A0F1C
- Surface: #141B2D
- Card surface: #1C2539
- Text primary: #F1F5F9
- Text secondary: #94A3B8
- Heading font: "Outfit" (Google Fonts)
- Body font: "DM Sans" (Google Fonts)
- Border radius: 16px cards, 12px buttons, 8px inputs
- All interactive elements must have hover/active transitions

## Rules
- Georgian + English bilingual UI
- Mobile-first responsive design
- No generic AI aesthetics — every screen must feel custom designed
